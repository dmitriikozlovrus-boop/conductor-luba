# Docs

Эта папка содержит проектную документацию по реализации системы: архитектуру, модели данных, описания сервисов, use cases, roadmap, архитектурные решения и общие технические материалы.

Используй `docs/` для документов, которые описывают, как система реализуется, развивается, эксплуатируется или планируется.
