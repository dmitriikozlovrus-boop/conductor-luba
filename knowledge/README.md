# Knowledge

Папка содержит устойчивые знания AI OS.

## Основные разделы

- `vision/` — видение и назначение системы.
- `architecture/` — устойчивые архитектурные принципы и стандарты.
- `entities/` — определения сущностей.
- `classification/` — правила выбора между сущностями.
- `development/` — стандарты разработки.
- `agents/` — реестр агентов, ролей и общих принципов агентной архитектуры.

## Правило

Если документ описывает устойчивое правило системы — он относится к `knowledge/`.

Если документ описывает проектное, техническое или продуктовое решение — он относится к `docs/`.
