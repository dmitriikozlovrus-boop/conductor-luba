# Knowledge

Данная папка содержит базовые знания системы AI OS.

## Structure

- entities/ — определения сущностей системы.
- classification/ — правила выбора между сущностями.

## Usage

Entity-документы отвечают на вопрос:

"Что такое данная сущность?"

Classification Rules отвечают на вопрос:

"Как выбирать между похожими сущностями?"

При конфликте между Entity Definition и Classification Rules приоритет имеет Classification Rules.
