# Project

## Описание

Project — это ограниченная по цели инициатива, объединяющая связанные задачи, исследования, проблемы, коммуникации и другие сущности для достижения конкретного результата.

Project существует для организации работы вокруг общей цели.

---

## Основная цель

Объединять связанные сущности в единый контекст и обеспечивать управление достижением конкретного результата.

---

## Относится

- рабочие проекты;
- бизнес-проекты;
- личные проекты;
- семейные проекты;
- продукты;
- сервисы;
- инициативы;
- мероприятия с конкретной целью;
- программы развития;
- проекты по улучшению системы;
- проекты с набором связанных действий и результатов.

---

## Не относится

- отдельные задачи;
- отдельные исследования;
- отдельные проблемы;
- отдельные идеи;
- контакты;
- компании;
- коммуникации;
- календарные события;
- темы знаний.

---

## Связанные сущности

- Task;
- Study;
- Problem;
- Event;
- Communication;
- Contact;
- Company;
- Topic;
- Stream;
- System Issue;
- Improvement.

---

## Ключевые признаки

- существует конкретная цель или результат;
- проект объединяет несколько связанных сущностей;
- проект имеет жизненный цикл;
- проект может содержать задачи, исследования, проблемы и коммуникации;
- завершение проекта приводит к достижению определенного результата.
